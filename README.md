# Covid-19-Analysis
📌 Project Overview

This project focuses on analyzing COVID-19 data in India using Microsoft Power BI. The objective is to transform raw data into meaningful insights through interactive dashboards and visualizations. It highlights trends in confirmed cases, recoveries, and deaths across different states over time.

🎯 Objectives
Analyze COVID-19 case trends across India
Compare state-wise performance in terms of confirmed, recovered, and death cases
Build interactive dashboards for better data interpretation
Support data-driven insights using visualization techniques
🛠️ Tools & Technologies
Power BI (Data Visualization & Dashboarding)
Power Query (Data Cleaning & Transformation)
DAX (Custom Measures & Calculations)
🧹 Data Preparation
Cleaned dataset using Power Query
Replaced missing values (e.g., “-”) with 0 in key columns
Structured data for accurate analysis and visualization
📈 Key Visualizations
1. Daily Case Trend
Line chart showing daily confirmed cases over time
Helps identify peaks and growth patterns
2. Top 10 States by Confirmed Cases
Bar chart highlighting states with highest cases
Uses Top-N filtering for focused insights
3. State-wise Death Count
Comparison of death counts across states
Identifies high-risk regions
4. Indian vs Foreign Cases
Stacked bar chart comparing case categories
Enables clear category-based comparison
5. Cumulative Deaths Over Time
Running total of deaths using DAX
Shows overall impact progression
6. Recovery vs Death Comparison
Clustered column chart comparing cured vs deaths
Evaluates healthcare response effectiveness
7. Recovery Trend
Line chart showing growth in recovered cases
Indicates improvement over time
8. Case Distribution by State
Donut chart representing case share by state
Visualizes contribution of each state
📊 Key Insights
COVID-19 cases showed significant growth during specific periods
Certain states consistently recorded higher confirmed and death counts
Recovery trends improved over time, reflecting better healthcare response
Distribution of cases varied widely across states
